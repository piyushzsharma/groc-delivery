# Groc - Grocery Delivery Web App

[![Deploy to Vercel](https://vercelbadge.vercel.app/api/piyushzsharma/groc-delivery)](https://groc-delivery.vercel.app/)

**Live Demo:** [https://groc-delivery.vercel.app/](https://groc-delivery.vercel.app/)

A modern, highly responsive React web application that simulates a complete grocery shopping and delivery experience. Built comprehensively from high-fidelity Figma designs, this project features robust global state management, strict TypeScript adherence, and fluid mobile-first layouts that scale beautifully seamlessly onto desktop screens.

## 🚀 Features

- **End-to-End Shopping Flow**: Dynamic product grids, category filtering, and detailed item views.
- **Persistent Cart Management**: Real-time quantity manipulation and price calculation powered by Zustand.
- **Complete Authentication Pipeline**: Onboarding, Login, Signup, and Location selection mock routing.
- **Fluid Checkout Experience**: Sliding cart modals, total calculations, and celebratory success states.
- **Responsive Architecture**: Edge-to-edge layouts engineered natively to switch gracefully between bottom-navigation mobile patterns and sticky desktop sidebars.

## 💻 Tech Stack

- **Framework**: React 18
- **Build Tool**: Vite (Lightning fast HMR)
- **Language**: TypeScript (Strict interface adherence)
- **Styling**: Tailwind CSS v4 (PostCSS integrated)
- **State Management**: Zustand
- **Routing**: React Router v6
- **Icons**: Lucide React

## 🛠️ Quick Start

**1. Clone the repository**
```bash
git clone https://github.com/piyushzsharma/groc-delivery.git
cd groc-delivery
```

**2. Install dependencies**
```bash
npm install
```

**3. Run the development server**
```bash
npm run dev
```

The application will instantly launch at `http://localhost:5173`. 

## 🌐 Deployment
This application is fully optimized for zero-config deployment on **Vercel** with integrated `vercel.json` rewrite logic to natively support client-side React Router behaviors. Simply import the repository to Vercel and it will host it continuously for free!
